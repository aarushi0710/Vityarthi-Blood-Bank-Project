The purpose of this project is to design a simple Blood Bank Management System
that helps register donors, store donor information, search for available blood groups,
and manage blood requests efficiently.

 Objectives
- Create a donor database
- Allow users to search donors by blood group
- Enable submission of blood requests
- Display active requests
Scope
This system is intended for small blood donation camps or communities
where quick access to donor information is required.

 
A functional console-based program that allows:
- Donor registration
- Donor listing
- Searching donors
